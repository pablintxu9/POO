# Catapulta Game

Este proyecto es un juego de catapultas en Python, donde los usuarios pueden construir su propia catapulta utilizando diferentes materiales y eliminar enemigos. El juego está diseñado utilizando programación orientada a objetos y se organiza en varios módulos para facilitar su mantenimiento y expansión.

## Estructura del Proyecto

El proyecto tiene la siguiente estructura de archivos:

```
catapulta-game
├── catapulta
│   ├── __init__.py
│   ├── materiales.py
│   ├── catapulta.py
│   ├── proyectil.py
│   ├── enemigos.py
│   ├── interfaz.py
│   └── utils.py
├── tests
│   ├── __init__.py
│   └── test_catapulta.py
├── main.py
├── requirements.txt
├── pyproject.toml
├── .gitignore
└── README.md
```

## Instalación

Para instalar las dependencias necesarias, utiliza el siguiente comando:

```
pip install -r requirements.txt
```

## Uso

1. Ejecuta el archivo `main.py` para iniciar el juego.
2. Sigue las instrucciones en pantalla para seleccionar los materiales y construir tu catapulta.
3. Lanza proyectiles y elimina enemigos para ganar puntos.
